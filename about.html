E D U C A T I O N

In academic wise, I just finished two group projects this year and both required to build applications using React connected it to firebase. The first project was building a prototype application for the company called FML, our team aimed to build a training online schedule for FML's training course for both trainers and schedulers. Whereas, the second project was to create a weather app with unique features that current existing weather apps don't have. Our team created both projects as browser applications so that the system will be highly accessible to all end-users through several devices and browsing platforms. We have ensured that the user interface is not compromised when switching between mobile and desktop devices to allow a seamless transition for users switching between portable and non-portable devices.

I enjoy designing and developing websites to improve my web development skills. My first experience in web programming was last year where I learnt how to build a basic portfolio website for myself and I enjoyed it because I like designing and being creative. Even though I have limited experiences in web programming, I’m willing to practice and research by myself in extra work to gain better understanding and knowledge and skills on that so I can achieve a better website application.
